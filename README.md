> A curated list of high-quality resources, tools, sensors, and case studies for using Lidar (Light Detection and Ranging) technology on drones.

Lidar-equipped drones are used in applications such as surveying, mapping, forestry, agriculture, construction, and environmental monitoring. This list is intended for UAV engineers, drone operators, researchers, and professionals in the geospatial and remote sensing fields.

## General Resources

- [Introduction to Lidar for Drones (DroneDeploy)](https://www.dronedeploy.com/blog/lidar-drone-surveying/) — Overview of Lidar applications with drones.
- [USGS Lidar Basics](https://www.usgs.gov/core-science-systems/ngp/3dep/lidar-basics) — Foundational concepts of Lidar technology.
- [Lidar 101 (Velodyne)](https://velodynelidar.com/what-is-lidar/) — Basic explanation of how Lidar works.

## Lidar Sensors for Drones

- [Velodyne Lidar](https://velodynelidar.com/) — Widely used Lidar sensors suitable for UAV integration.
- [Ouster Lidar](https://ouster.com/) — High-resolution digital Lidar sensors for drones and robotics.
- [RIEGL UAV Lidar](https://www.riegl.com/products/uav-lidar-scanning/) — High-precision Lidar systems optimized for UAV use.
- [Livox Lidar](https://www.livoxtech.com/) — Affordable and lightweight Lidar solutions for drone applications.

## Drone Lidar Platforms

- [DJI L1 Lidar Module](https://www.dji.com/zenmuse-l1) — Integrated Lidar system for DJI enterprise drones.
- [Microdrones Lidar Solutions](https://www.microdrones.com/en/products/mdsolutions/) — Turnkey drone + Lidar packages.
- [YellowScan Lidar Drones](https://www.yellowscan-lidar.com/) — Professional UAV Lidar systems for surveying.

## Software & Processing Tools

- [CloudCompare](https://www.cloudcompare.org/) — Open-source 3D point cloud and mesh processing software.
- [LAStools](https://rapidlasso.com/lastools/) — Popular tools for Lidar data processing and management.
- [Pix4Dmapper](https://www.pix4d.com/product/pix4dmapper-photogrammetry-software) — Photogrammetry software that supports Lidar integration.
- [TerraSolid](https://www.terrasolid.com/) — Software suite for processing airborne and UAV Lidar data.

## Datasets & Research

- [OpenTopography](https://opentopography.org/) — Public Lidar datasets including drone-collected data.
- [USGS 3DEP Lidar Data](https://www.usgs.gov/3d-elevation-program) — Nationwide elevation data using Lidar.
- [UAV Lidar Research Papers (Google Scholar)](https://scholar.google.com/scholar?q=uav+lidar) — Academic research on drone Lidar systems.
